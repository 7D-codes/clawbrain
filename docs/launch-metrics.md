## RED FLAGS (Investigate Immediately)

- [ ] DAU drops 20%+ for 3 days straight
- [ ] Waitlist conversion drops below 5%
- [ ] Churn rate exceeds 10%/month
- [ ] Uptime drops below 99%
- [ ] Support tickets spike
- [ ] Negative sentiment in community
