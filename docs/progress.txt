## 2026-02-05 - CB-002 Task Manager Skill Complete

### Completed
1. ✅ **Created SKILL.md** — Documentation for the task-manager skill
2. ✅ **Created package.json** — Skill manifest with dependencies (uuid, js-yaml, gray-matter)
3. ✅ **Created index.js** — Full skill implementation with:
   - `createTask()` — Creates task with YAML frontmatter
   - `listTasks()` — Lists all tasks with optional project/status filters
   - `getTask()` — Gets single task by ID or slug
   - `updateTask()` — Updates task fields (title, status, project)
   - `deleteTask()` — Deletes task by ID or slug
   - Natural language command parsing for all operations

### Features
- **File Format**: Markdown with YAML frontmatter following ARCHITECTURE.md schema
- **Storage**: `~/clawdbrain/tasks/task-{uuid}.md`
- **Commands**:
  - `create task: "<title>" [in project <project>]`
  - `list tasks [in project <project>] [with status <status>]`
  - `update task <id|slug> status to <todo|in-progress|done>`
  - `update task <id|slug> title to "<new title>"`
  - `delete task <id|slug>`
- **Project Associations**: Tasks support project field for filtering
- **Slug Generation**: Auto-generated from title (kebab-case, 50 char limit)
- **Timestamps**: ISO 8601 created/updated timestamps

### Testing
- ✅ Created and ran test.js — all 10 tests passed
